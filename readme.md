# psychopy 实现简易SSVEP视觉刺激

psychopy是专业的心理学刺激库，

提供了**图形化编程界面**，极大缩短编写刺激界面的时间。

可以实现对每一帧的操作，

符合ssvep高时间精度刺激的要求。

## 使用方法
- 安装psychopy
- 运行ppp_lastrun.py
